Política de Privacidade

Introdução

Esta Política de Privacidade descreve como tratamos e protegemos os dados pessoais dos nossos usuários, em conformidade com a Lei Geral de Proteção de Dados (LGPD). Nosso compromisso é garantir a privacidade e a segurança das informações que coletamos, incluindo o número de CPF.

Coleta e Uso do CPF

O número de CPF é coletado exclusivamente para fins de autenticação e identificação do usuário em nosso sistema. Utilizamos o CPF para garantir que cada usuário tenha uma conta única e segura, evitando fraudes e acessos não autorizados.

Base Legal para Tratamento de Dados

O tratamento do CPF do usuário é realizado com base no consentimento expresso fornecido pelo próprio usuário no momento do cadastro. Este consentimento pode ser revogado a qualquer momento, mediante solicitação, conforme as disposições da LGPD.

Armazenamento e Proteção dos Dados

Os dados pessoais, incluindo o CPF, são armazenados de maneira segura, com medidas técnicas e organizacionais adequadas para prevenir acessos não autorizados, perda, destruição ou divulgação de informações. Utilizamos criptografia e outras práticas de segurança para proteger esses dados.

Compartilhamento de Dados

Não compartilhamos o número de CPF ou outras informações pessoais dos usuários com terceiros, a menos que seja necessário para cumprir uma obrigação legal ou regulatória, ou com o consentimento prévio do usuário.

Direitos dos Titulares dos Dados

Os usuários têm o direito de acessar, corrigir ou solicitar a exclusão de seus dados pessoais a qualquer momento. Para exercer esses direitos, os usuários devem entrar em contato conosco por meio dos canais de atendimento disponíveis.

Retenção de Dados

Os dados pessoais, incluindo o CPF, são mantidos pelo período necessário para cumprir as finalidades para as quais foram coletados, ou conforme exigido por lei. Após o término desse período, os dados serão excluídos ou anonimizados de forma segura.

Alterações na Política de Privacidade

Podemos atualizar esta Política de Privacidade periodicamente para refletir mudanças em nossas práticas ou nas leis aplicáveis. Recomendamos que os usuários revisem esta página regularmente para se manterem informados sobre como estamos protegendo suas informações.
